Sinais exponenciais de tempo discreto¶

Renato Naville Watanabe

Exponencial real¶

Em sinais discretos a exponencial real é definida da seguinte maneira:

$$x[n] = Ae^{\beta n}$$

Embora a expressão acima esteja correta, para sinais de tempo discreto é comum definirmos $\alpha = e^\beta$. Com isso, temos:

$$x[n] = A\alpha^n$$

Com $\alpha$ e $A$ reais.

Exponencial complexa¶

Da mesma forma que no caso contínuo, podemos definir um sinal com o expoente complexo:

$$x[n] = e^{j\omega_0 n}$$

Pela relação de Euler-Moivre, este sinal está totalmente relacionado com o cosseno:

$$x[n] = A\cos(\omega_0 n + \theta) = A\frac{e^{j(\omega_0 n + \theta)} + e^{-j(\omega_0 n + \theta)}}{2}$$

Abaixo, são mostrados diversos sinais cosseno, com diferentes frequências (as frequências sobem ao longo dos gráficos, da esquerda para a direita e de cima para baixo).

Uma outra diferença em relação ao cosseno (e exponencial complexa) de tempo contínuo é que nem sempre o sinal vai ser periódico. Para percebermos isso, vamos usar a definição de sinal periódico:

$$e^{j\omega_0 n} = e^{j\omega_0 (n+N)} = e^{j\omega_0 n}e^{j\omega_0 N}$$

A expressão acima só vai ter os dois lados iguais se:

$$ e^{j\omega_0 N} = 1$$

Isso só vai acontecer se existir algum $N$ inteiro que faça $\omega_0 N=2\pi$. Se existir, esse $N$ é que é o período fundamental do sinal. Mas, para existir, $\frac{\omega_0}{2\pi}$ precisa ser racional. Se essa divisão não for racional, o sinal não será periódico. Por exemplo, para:
